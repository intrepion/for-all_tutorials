# Frameworks

This folder is for reusable framework-specific setup, not for the main tutorial matrix.

Suggested contents per framework:

- starter template
- conventions and notes
- shared scripts
- framework-specific troubleshooting

Example:

```text
frameworks/rust-leptos-ssr/
frameworks/elixir-phoenix-liveview/
frameworks/python-fastapi-jinja-htmx/
```

Keep these folders generic so multiple tutorial apps can reuse them.
