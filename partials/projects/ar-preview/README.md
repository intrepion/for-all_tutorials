## Purpose

`ar-preview` exists to teach:

- enabling AR mode for a 3D viewer
- selecting a canonical AR scale mode
- handling supported and unsupported device capabilities gracefully
- separating user-facing fallback rules from adapter-managed platform wiring
- keeping adapters thin while the core owns AR messaging and scale defaults

This project follows the shared workflow and output model in [Projects](../README.md#shared-workflow) and [Projects](../README.md#shared-output-model).

The supported compiled tutorial outputs for this project are declared in [Manifest](manifest.yaml).
