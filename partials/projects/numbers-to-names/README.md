## Purpose

`numbers-to-names` exists to teach:

- parsing a prompted month number into a numeric value
- mapping a numeric value to a named month
- branching between a successful lookup and an error case
- separating month lookup from output formatting
- keeping adapters thin while the core owns the mapping and error rules

This project follows the shared workflow and output model in [Projects](../README.md#shared-workflow) and [Projects](../README.md#shared-output-model).

The supported compiled tutorial outputs for this project are declared in [Manifest](manifest.yaml).
