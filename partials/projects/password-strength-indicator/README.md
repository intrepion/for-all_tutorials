## Purpose

`password-strength-indicator` exists to teach:

- scoring one text input by length and character categories
- separating score calculation, strength classification, and output formatting
- handling clearly defined category rules that fully cover the input space
- keeping adapters thin while the core owns the strength rules

This project follows the shared workflow and output model in [Projects](../README.md#shared-workflow) and [Projects](../README.md#shared-output-model).

The supported compiled tutorial outputs for this project are declared in [Manifest](manifest.yaml).
