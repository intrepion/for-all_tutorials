## Purpose

`printing-quotes` exists to teach:

- combining two text inputs into one formatted output
- surrounding user-provided text with literal quotation marks
- preserving author and quote text exactly instead of rewriting it
- keeping adapters thin while the core owns the formatting rule

This project follows the shared workflow and output model in [Projects](../README.md#shared-workflow) and [Projects](../README.md#shared-output-model).

The supported compiled tutorial outputs for this project are declared in [Manifest](manifest.yaml).
