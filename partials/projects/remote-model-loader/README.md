## Purpose

`remote-model-loader` exists to teach:

- loading one remote 3D model asset
- calculating and displaying precise progress percentages
- handling ready, loading, and failed states explicitly
- preventing interaction until the model is fully ready
- keeping adapters thin while the core owns progress and error messaging rules

This project follows the shared workflow and output model in [Projects](../README.md#shared-workflow) and [Projects](../README.md#shared-output-model).

The supported compiled tutorial outputs for this project are declared in [Manifest](manifest.yaml).
