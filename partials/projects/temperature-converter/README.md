## Purpose

`temperature-converter` exists to teach:

- parsing a conversion choice and a numeric temperature at the adapter boundary
- branching between Fahrenheit-to-Celsius and Celsius-to-Fahrenheit formulas
- applying conversion formulas with deterministic rounding
- separating conversion calculation from output formatting
- keeping adapters thin while the core owns the branch and conversion rules

This project follows the shared workflow and output model in [Projects](../README.md#shared-workflow) and [Projects](../README.md#shared-output-model).

The supported compiled tutorial outputs for this project are declared in [Manifest](manifest.yaml).
