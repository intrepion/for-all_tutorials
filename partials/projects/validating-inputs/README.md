## Purpose

`validating-inputs` exists to teach:

- validating several fields with small focused functions
- separating field-level validation from error aggregation and output formatting
- collecting multiple validation errors instead of stopping at the first one
- keeping adapters thin while the core owns the validation rules

This project follows the shared workflow and output model in [Projects](../README.md#shared-workflow) and [Projects](../README.md#shared-output-model).

The supported compiled tutorial outputs for this project are declared in [Manifest](manifest.yaml).
