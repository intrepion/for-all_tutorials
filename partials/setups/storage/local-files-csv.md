# Local Files CSV

Use `local-files-csv` when an adapter persists application data in one or more local CSV files.

## Default Fit

- command-line adapters
- desktop or other local single-user tools
- export-oriented workflows where CSV is the canonical stored shape

## Default Compatibility Rule

Do not combine `local-files-csv` with web adapters unless a project explicitly requires local CSV persistence on the server.
