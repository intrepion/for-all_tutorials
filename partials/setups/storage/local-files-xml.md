# Local Files XML

Use `local-files-xml` when an adapter persists application data in one or more local XML files.

## Default Fit

- command-line adapters
- desktop or other local single-user tools
- projects that must interoperate with XML-shaped local data

## Default Compatibility Rule

Do not combine `local-files-xml` with web adapters unless a project explicitly requires local XML persistence on the server.
