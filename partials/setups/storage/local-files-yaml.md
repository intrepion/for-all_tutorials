# Local Files YAML

Use `local-files-yaml` when an adapter persists application data in one or more local YAML files.

## Default Fit

- command-line adapters
- desktop or other local single-user tools
- projects that want a human-editable structured file format

## Default Compatibility Rule

Do not combine `local-files-yaml` with web adapters unless a project explicitly requires local YAML persistence on the server.
