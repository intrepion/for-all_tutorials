# Adding Numbers

Project home for the twenty-sixth project app in the curriculum.

## Contents

- [Spec](spec/README.md)
- [Tutorial](tutorial/README.md)

## Purpose

`adding-numbers` exists to teach:

- summing a fixed list of numeric inputs
- using repeated input collection in the adapter
- separating numeric accumulation from output formatting
- keeping adapters thin while the core owns the addition rules

This project follows the shared workflow and output model in [Projects](../README.md#shared-workflow) and [Projects](../README.md#shared-output-model).
