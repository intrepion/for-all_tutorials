# Anagram Checker

Project home for the twenty-second project app in the curriculum.

## Contents

- [Spec](spec/README.md)
- [Tutorial](tutorial/README.md)

## Purpose

`anagram-checker` exists to teach:

- comparing two strings by their exact character counts
- separating anagram detection from output formatting
- preserving both original inputs exactly in the final message
- keeping adapters thin while the core owns the anagram rules

This project follows the shared workflow and output model in [Projects](../README.md#shared-workflow) and [Projects](../README.md#shared-output-model).
