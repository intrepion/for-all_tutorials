## Contents

- [Tutorial](tutorial/README.md)

## Purpose

`bmi-calculator` exists to teach:

- parsing prompted height and weight into numeric values
- applying the BMI formula with deterministic rounding
- classifying the rounded BMI against a fixed healthy range
- branching between ideal, underweight, and overweight messages
- separating BMI calculation, classification, and output formatting
- keeping adapters thin while the core owns the formula and range rules

This project follows the shared workflow and output model in [Projects](../README.md#shared-workflow) and [Projects](../README.md#shared-output-model).
