# Computing Statistics

Project home for the `computing-statistics` curriculum project.

## Contents

- [Spec](spec/README.md)
- [Tutorial](tutorial/README.md)

## Purpose

`computing-statistics` exists to teach:

- collecting a dynamic list of numeric response times
- ending collection with a sentinel value
- computing mean, minimum, maximum, and standard deviation
- separating statistical calculations and report formatting from input parsing and control flow
- keeping adapters thin while the core owns the statistics rules

This project follows the shared workflow and output model in [Projects](../README.md#shared-workflow) and [Projects](../README.md#shared-output-model).
