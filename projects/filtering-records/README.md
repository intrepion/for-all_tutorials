# Filtering Records

Project home for the `filtering-records` curriculum project.

## Contents

- [Spec](spec/README.md)
- [Tutorial](tutorial/README.md)

## Purpose

`filtering-records` exists to teach:

- working with a fixed canonical list of employee records
- locating records by searching first and last name fields
- preserving source order while filtering and then applying a canonical sort
- rendering filtered records in the same table-style report as the sorting exercise
- separating dataset construction, filtering, sorting, and formatting
- keeping adapters thin while the core owns the record-search rules

This project follows the shared workflow and output model in [Projects](../README.md#shared-workflow) and [Projects](../README.md#shared-output-model).
