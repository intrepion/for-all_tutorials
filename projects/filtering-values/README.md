# Filtering Values

Project home for the `filtering-values` curriculum project.

## Contents

- [Spec](spec/README.md)
- [Tutorial](tutorial/README.md)

## Purpose

`filtering-values` exists to teach:

- collecting a space-separated list of numbers
- preserving input order while filtering that list
- selecting only the even values from a larger collection
- separating list filtering and output formatting from input parsing
- keeping adapters thin while the core owns the filtering rule

This project follows the shared workflow and output model in [Projects](../README.md#shared-workflow) and [Projects](../README.md#shared-output-model).
