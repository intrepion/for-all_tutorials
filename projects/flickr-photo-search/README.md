# Flickr Photo Search

Project home for the `flickr-photo-search` curriculum project.

## Contents

- [Spec](spec/README.md)
- [Tutorial](tutorial/README.md)

## Purpose

`flickr-photo-search` exists to teach:

- querying Flickr's public photo feed with a search string
- parsing feed items into structured photo results
- preserving source order from a live API response
- separating HTTP transport from deterministic response parsing
- building a thin graphical adapter that displays actual photographs
- keeping adapters thin while the core owns the response-shape and display-item rules

This project follows the shared workflow and output model in [Projects](../README.md#shared-workflow) and [Projects](../README.md#shared-output-model).
