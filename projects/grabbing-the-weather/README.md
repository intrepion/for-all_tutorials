# Grabbing the Weather

Project home for the `grabbing-the-weather` curriculum project.

## Contents

- [Spec](spec/README.md)
- [Tutorial](tutorial/README.md)

## Purpose

`grabbing-the-weather` exists to teach:

- fetching live JSON from a weather API in the adapter
- parsing the current-weather response into a small deterministic report
- rounding a live temperature value to a whole Fahrenheit degree
- formatting a simple two-line weather report
- separating HTTP transport from response parsing and output formatting
- keeping adapters thin while the core owns the response-shape and report rules

This project follows the shared workflow and output model in [Projects](../README.md#shared-workflow) and [Projects](../README.md#shared-output-model).
