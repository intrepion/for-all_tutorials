# Guess the Number Game

Project home for the `guess-the-number-game` curriculum project.

## Contents

- [Spec](spec/README.md)
- [Tutorial](tutorial/README.md)

## Purpose

`guess-the-number-game` exists to teach:

- mapping difficulty levels to numeric ranges
- comparing guesses against a hidden target number
- tracking guess counts through a game loop
- keeping randomness and replay flow in the adapter while the core owns the game rules

This project follows the shared workflow and output model in [Projects](../README.md#shared-workflow) and [Projects](../README.md#shared-output-model).
