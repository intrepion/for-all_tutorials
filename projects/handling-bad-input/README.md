# Handling Bad Input

Project home for the twenty-seventh project app in the curriculum.

## Contents

- [Spec](spec/README.md)
- [Tutorial](tutorial/README.md)

## Purpose

`handling-bad-input` exists to teach:

- validating one prompted value before using it in a calculation
- retrying in the adapter until a valid value is entered
- separating input parsing, calculation, and output formatting
- keeping adapters thin while the core owns the validation and rule-of-72 logic

This project follows the shared workflow and output model in [Projects](../README.md#shared-workflow) and [Projects](../README.md#shared-output-model).
