# Karvonen Heart Rate

Project home for the twenty-ninth project app in the curriculum.

## Contents

- [Spec](spec/README.md)
- [Tutorial](tutorial/README.md)

## Purpose

`karvonen-heart-rate` exists to teach:

- applying the Karvonen formula in a small core function
- generating a stepped table from 55% through 95%
- rounding heart-rate results to whole beats per minute
- keeping adapters thin while the core owns the table-generation rules

This project follows the shared workflow and output model in [Projects](../README.md#shared-workflow) and [Projects](../README.md#shared-output-model).
