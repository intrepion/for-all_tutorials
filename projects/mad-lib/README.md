# Mad Lib

Project home for the fourth project app in the curriculum.

## Contents

- [Spec](spec/README.md)
- [Tutorial](tutorial/README.md)

## Purpose

`mad-lib` exists to teach:

- combining several named inputs into one fixed story template
- preserving user-provided words exactly while placing them into a sentence
- making one small formatting function own the story rule
- keeping adapters thin while the core owns the sentence construction

This project follows the shared workflow and output model in [Projects](../README.md#shared-workflow) and [Projects](../README.md#shared-output-model).
