# Magic 8 Ball

Project home for the `magic-8-ball` curriculum project.

## Contents

- [Spec](spec/README.md)
- [Tutorial](tutorial/README.md)

## Purpose

`magic-8-ball` exists to teach:

- keeping a fixed set of possible responses in the core logic
- choosing one response at random in the adapter
- separating response lookup from random selection and input/output
- keeping adapters thin while the core owns the canonical response set

This project follows the shared workflow and output model in [Projects](../README.md#shared-workflow) and [Projects](../README.md#shared-output-model).
