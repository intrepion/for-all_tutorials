## Contents

- [Tutorial](tutorial/README.md)

## Purpose

`movie-recommendations` exists to teach:

- prompting for a movie search query
- retrieving movie metadata and audience score data from an approved movie-data provider
- parsing movie details into a deterministic report
- formatting a multi-line movie summary
- generating recommendation guidance from audience-score thresholds
- separating provider access from parsing, formatting, and recommendation logic
- keeping adapters thin while the core owns the response-shape and recommendation rules

This project follows the shared workflow and output model in [Projects](../README.md#shared-workflow) and [Projects](../README.md#shared-output-model).
