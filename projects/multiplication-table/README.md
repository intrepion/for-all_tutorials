# Multiplication Table

Project home for the twenty-eighth project app in the curriculum.

## Contents

- [Spec](spec/README.md)
- [Tutorial](tutorial/README.md)

## Purpose

`multiplication-table` exists to teach:

- generating a fixed two-dimensional numeric table
- using nested iteration in the core logic
- separating table generation from output formatting
- keeping adapters thin while the core owns the table rules

This project follows the shared workflow and output model in [Projects](../README.md#shared-workflow) and [Projects](../README.md#shared-output-model).
