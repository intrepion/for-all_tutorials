# Paint Calculator

Project home for the ninth project app in the curriculum.

## Contents

- [Spec](spec/README.md)
- [Tutorial](tutorial/README.md)

## Purpose

`paint-calculator` exists to teach:

- converting two prompted room dimensions into numeric values
- calculating ceiling area in square feet
- converting square feet into a whole-gallon paint purchase count
- rounding up instead of using standard rounding rules
- separating paint calculation from output formatting
- keeping adapters thin while the core owns the coverage and rounding rules

This project follows the shared workflow and output model in [Projects](../README.md#shared-workflow) and [Projects](../README.md#shared-output-model).
