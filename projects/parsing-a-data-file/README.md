# Parsing a Data File

Project home for the `parsing-a-data-file` curriculum project.

## Contents

- [Spec](spec/README.md)
- [Tutorial](tutorial/README.md)

## Purpose

`parsing-a-data-file` exists to teach:

- reading canonical CSV lines from a file in the adapter
- parsing those lines into structured records
- preserving input order while transforming CSV text into typed fields
- formatting parsed records as a canonical table report
- separating CSV parsing and table formatting from file input and output
- keeping adapters thin while the core owns the parsing and display rules

This project follows the shared workflow and output model in [Projects](../README.md#shared-workflow) and [Projects](../README.md#shared-output-model).
