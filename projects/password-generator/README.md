# Password Generator

Project home for the `password-generator` curriculum project.

## Contents

- [Spec](spec/README.md)
- [Tutorial](tutorial/README.md)

## Purpose

`password-generator` exists to teach:

- turning password requirements into a deterministic generation plan
- combining letter, digit, and special-character pools
- keeping random character selection and shuffling in the adapter
- separating password rules and output formatting from random generation
- keeping adapters thin while the core owns the password-generation rules

This project follows the shared workflow and output model in [Projects](../README.md#shared-workflow) and [Projects](../README.md#shared-output-model).
