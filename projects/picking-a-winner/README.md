# Picking a Winner

Project home for the `picking-a-winner` curriculum project.

## Contents

- [Spec](spec/README.md)
- [Tutorial](tutorial/README.md)

## Purpose

`picking-a-winner` exists to teach:

- collecting a dynamic list of contestant names from user input
- ending list entry on a blank sentinel value
- selecting one winner by position after the adapter chooses a random value
- separating winner lookup from random number generation and input/output
- keeping adapters thin while the core owns the deterministic winner-selection rules

This project follows the shared workflow and output model in [Projects](../README.md#shared-workflow) and [Projects](../README.md#shared-output-model).
