# Pushing Notes to Firebase

Project home for the `pushing-notes-to-firebase` curriculum project.

## Contents

- [Spec](spec/README.md)
- [Tutorial](tutorial/README.md)

## Purpose

`pushing-notes-to-firebase` exists to teach:

- building a command-line app with multiple commands
- saving note data through the Firebase Realtime Database REST API
- reading back a collection of saved notes from a JSON response
- parsing note records out of a keyed JSON object
- formatting note reports in descending date order
- separating REST transport and command handling from deterministic note logic
- keeping adapters thin while the core owns the note-shape and report rules

This project follows the shared workflow and output model in [Projects](../README.md#shared-workflow) and [Projects](../README.md#shared-output-model).
