# Retirement Calculator

Project home for the sixth project app in the curriculum.

## Contents

- [Spec](spec/README.md)
- [Tutorial](tutorial/README.md)

## Purpose

`retirement-calculator` exists to teach:

- combining prompted numeric input with the current year from the runtime
- calculating years-left and retirement-year values
- separating date arithmetic from output formatting
- keeping adapters thin while the core owns the retirement calculation and report rules

This project follows the shared workflow and output model in [Projects](../README.md#shared-workflow) and [Projects](../README.md#shared-output-model).
