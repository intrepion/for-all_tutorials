# Simple Math

Project home for the fifth project app in the curriculum.

## Contents

- [Spec](spec/README.md)
- [Tutorial](tutorial/README.md)

## Purpose

`simple-math` exists to teach:

- converting two plain-text inputs into numeric values at the adapter boundary
- performing basic arithmetic in the core logic
- separating numeric calculation from output formatting
- keeping adapters thin while the core owns the arithmetic and report rules

This project follows the shared workflow and output model in [Projects](../README.md#shared-workflow) and [Projects](../README.md#shared-output-model).
