## Contents

## Purpose

`trivia-app` exists to teach:

- reading a question bank from a local data file
- parsing structured trivia questions into deterministic records
- separating question content from random question order and random answer order
- checking answers against the correct response
- tracking a running score until the player misses a question
- keeping adapters thin while the core owns the question-bank and answer-checking rules

This project follows the shared workflow and output model in [Projects](../README.md#shared-workflow) and [Projects](../README.md#shared-output-model).
