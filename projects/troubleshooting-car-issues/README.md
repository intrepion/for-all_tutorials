# Troubleshooting Car Issues

Project home for the twenty-first project app in the curriculum.

## Contents

- [Spec](spec/README.md)
- [Tutorial](tutorial/README.md)

## Purpose

`troubleshooting-car-issues` exists to teach:

- walking a user through a nested yes/no decision tree
- separating decision-tree resolution from message formatting
- handling only the questions required by the current branch
- keeping adapters thin while the core owns the troubleshooting rules

This project follows the shared workflow and output model in [Projects](../README.md#shared-workflow) and [Projects](../README.md#shared-output-model).
