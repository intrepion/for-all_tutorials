# URL Shortener

Project home for the `url-shortener` curriculum project.

## Contents

- [Spec](spec/README.md)
- [Tutorial](tutorial/README.md)

## Purpose

`url-shortener` exists to teach:

- building a full-stack web app with form submission, redirect behavior, and a stats page
- generating short local URL paths from adapter-chosen short codes
- persisting short-link records in a local data store
- incrementing visit counts through repeated redirects
- formatting deterministic statistics output from a short-link record
- separating routing, persistence, and redirect behavior from core short-link logic
- keeping adapters thin while the core owns the short-link rules

This project follows the shared workflow and output model in [Projects](../README.md#shared-workflow) and [Projects](../README.md#shared-output-model).
