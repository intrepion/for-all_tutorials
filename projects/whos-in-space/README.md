# Who's in Space?

Project home for the `whos-in-space` curriculum project.

## Contents

- [Spec](spec/README.md)
- [Tutorial](tutorial/README.md)

## Purpose

`whos-in-space` exists to teach:

- fetching JSON from a live HTTP API in the adapter
- parsing API response data into structured records
- preserving source order from the API response
- formatting a tabular report from parsed response data
- separating network access, parsing, and formatting responsibilities
- keeping adapters thin while the core owns the response-shape and table-format rules

This project follows the shared workflow and output model in [Projects](../README.md#shared-workflow) and [Projects](../README.md#shared-output-model).
