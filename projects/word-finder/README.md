# Word Finder

Project home for the `word-finder` curriculum project.

## Contents

- [Spec](spec/README.md)
- [Tutorial](tutorial/README.md)

## Purpose

`word-finder` exists to teach:

- reading a source text file in the adapter
- transforming file contents with one deterministic replacement rule
- replacing exact word occurrences while preserving all other text
- writing transformed contents to a new file
- separating text transformation from file input and output
- keeping adapters thin while the core owns the replacement rule

This project follows the shared workflow and output model in [Projects](../README.md#shared-workflow) and [Projects](../README.md#shared-output-model).
