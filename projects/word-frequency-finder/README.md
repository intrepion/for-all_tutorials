# Word Frequency Finder

Project home for the `word-frequency-finder` curriculum project.

## Contents

- [Spec](spec/README.md)
- [Tutorial](tutorial/README.md)

## Purpose

`word-frequency-finder` exists to teach:

- reading a text file in the adapter
- tokenizing text into words in the core
- counting exact word frequencies
- preserving first-appearance order while building a histogram
- formatting the histogram as canonical output lines
- keeping adapters thin while the core owns the tokenization, counting, and histogram rules

This project follows the shared workflow and output model in [Projects](../README.md#shared-workflow) and [Projects](../README.md#shared-output-model).
