# Languages

Languages commonly used in the `.NET` ecosystem.

## Primary Languages

- `C#`
- `F#`
- `Visual Basic`

## Guidance

Choose the language that best matches the tutorial path you are building.

Language choice changes:

- syntax
- project templates
- testing syntax
- framework ergonomics

Language choice should not change:

- the project contract
- the TDD order in the project tutorial
- the repo-wide coverage expectations
