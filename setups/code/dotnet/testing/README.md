# Testing

Reusable unit-test setup guides for `.NET` tutorials.

## Contents

- [xUnit](xunit.md)
- [NUnit](nunit.md)
- [MSTest](mstest.md)
- [TUnit](tunit.md)

## Guidance

Choose one testing framework for a single run of a tutorial and keep it consistent for that run.

Testing-framework choice should affect:

- test-project scaffolding
- assertion syntax
- coverage workflow

It should not affect:

- the project contract
- the TDD order in the project tutorial
- the behavioral requirements of the project
