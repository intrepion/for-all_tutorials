# Tutorials

This folder contains the runnable apps.

Recommended layout:

```text
tutorials/<level>/<project>/<framework-id>/
```

Examples:

```text
tutorials/level-1/guestbook/rust-leptos-ssr/
tutorials/level-1/guestbook/elixir-phoenix-liveview/
tutorials/level-3/forum-discussion-board/go-echo-templates-htmx/
```

Every implementation should point back to a single spec in `specs/` so the comparison stays fair.
